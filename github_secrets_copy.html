🚀 الخطوة الثالثة: إضافة Secrets في GitHub

  1. اذهب إلى GitHub Repository
  2. انقر على Settings
  3. من القائمة الجانبية، اختر Secrets and variables → Actions
  4. انقر على New repository secret
  5. أضف Secret الأول:
    • Name: DROPLET_IP
    • Secret: انسخ من الأعلى
  6. أضف Secret الثاني:
    • Name: DROPLET_SSH_KEY
    • Secret: انسخ المفتاح الخاص كاملاً
